3D Surface & Contour Plotter

Interactive plotting tool for scientific data visualization

Instructions:

  • Data Entry: Enter X-axis values in the first row, Y-axis values in the first column
  • Copy/Paste: You can paste data directly from Excel or Google Sheets
  • Auto-Trim: Empty rows and columns are automatically ignored when plotting
  • Required: All data cells must contain numeric values (no empty cells in data region)
  • Plot Types: Choose between 3D surface plots and 2D contour plots
  • Controls: Adjust axis ranges, scaling, colors, and viewing angles using the controls panel
